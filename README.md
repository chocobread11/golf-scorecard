# Golf Score

A minimalist, offline-first golf scorecard web app.

Built for golfers who want:
- Fast scoring
- No accounts
- No ads
- No distractions

## Features
- 9 or 18 hole rounds
- Up to 4 players
- Big, readable scores
- Automatic start & end time
- Full scorecard image export
- Works offline (PWA-ready)

## Philosophy
This app is designed as a **tool**, not a platform.
No tracking, no sign-ups, no clutter.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- html-to-image
